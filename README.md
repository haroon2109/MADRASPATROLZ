# 🏎️ MADRASPATROLZ 2.0 | Elite Car Spotting Chennai

**LIVE DEMO: [madraspatrolz.vercel.app](https://madraspatrolz.vercel.app/)** 🌍✨

**Madraspatrolz 2.0** is an immersive, high-performance web platform dedicated to the supercar spotting community in Chennai, India. Built with a high-impact **Neo-Brutalist** design system, it captures the raw energy of the streets and the precision of world-class automotive engineering.

---

## 🚀 Experience the Hunt
The streets of Madras hide the world's most exclusive machines. From screaming naturally aspirated V12s to silent, cutting-edge hybrid hypercars, this platform brings the experience to your screen.

### 🔊 High-Octane Features
- **Rev-Counter Soundboard**: Trigger authentic exhaust notes from legendary cars like the Aventador SVJ, Ferrari 458, and Revuelto.
- **Exhaust Flame Overlay**: Visual "spitting fire" effects synced with audio redlines.
- **3D Tilt Gallery**: Interactive car cards that respond to your cursor with depth and perspective.
- **Cyber-Search**: Instant, real-time filtering for models, locations (ECR, OMR, Alwarpet), and tags.
- **Night Patrol Mode**: A deep-space dark theme with high-contrast neon accents, built for late-night enthusiasts.
- **Interactive Spec Sheets**: Technical data (HP, Heart, Warp speed) for the top spots.

---

## 🛠️ Built With
- **Language**: Vanilla JavaScript (ES6+)
- **Styling**: Modern CSS3 (Neo-Brutalist Architecture)
- **Structure**: Semantic HTML5
- **Interactions**: Custom Cursor & Magnetic UI

## 📦 Local Setup
Since this is a static project, no complex installation is required.

1. Clone the repository:
   ```bash
   git clone https://github.com/haroon2109/MADRASPATROLZ.git
   ```
2. Open `index.html` in any modern web browser.
3. (Optional) Deploy instantly via Vercel or Netlify by dragging the project folder.
